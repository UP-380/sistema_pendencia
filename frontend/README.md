# Frontend React - UP380 Gestão de Pendências

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

Acesse `http://localhost:3000`

## Build para Produção

```bash
npm run build
```

Os arquivos serão gerados em `../static/react-build/`

## Estrutura

- `src/pages/` - Páginas do sistema
- `src/components/` - Componentes reutilizáveis
- `src/contexts/` - Contextos React (Auth, etc)
- `src/services/` - Serviços (API, etc)
